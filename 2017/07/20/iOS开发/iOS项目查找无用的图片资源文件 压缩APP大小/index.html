iOS项目查找无用的图片资源文件 压缩APP大小

发表于 2017-07-20 | 分类于 iOS开发

应用GitHub地址

1. 下载项目并运行

2. 将项目所在文件夹拖入 Project Path (粉色框) 内

3. 点击右下角Search 红色边框内的搜索结果即为没有用到的图片资源文件

注意 搜索结果仅为项目中未直接用到的图片资源, 如果图片名为拼写得到, 应对图片进行筛选
